# Hotel-Information-Agent

## Overview

This project demonstrates the development of a grounded Hotel Information Agent using Azure AI Foundry and Azure OpenAI models.

The agent is designed to provide accurate, professional, and context-aware hotel support through prompt engineering, retrieval-based grounding, safety guardrails, and evaluation workflows.

## Features

- Hotel guest support
- Grounded knowledge retrieval
- Hallucination prevention
- Conversational AI
- Hospitality-focused prompt engineering
- Evaluation and safety testing
- Edge-case handling

## Technologies Used

- Azure AI Foundry
- Azure OpenAI
- AI Model: gpt-5-mini
- Prompt Engineering
- AI Evaluation Pipelines
- Retrieval-Augmented Generation (RAG)

## Evaluation Metrics

The agent was evaluated using:
- Groundedness
- Relevance
- Fluency
- Coherence
- Task Completion
- Intent Resolution
- Safety Evaluators
